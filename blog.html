Important FAQ !!

What is the Document Object Model (DOM)?

The DOM is like a blueprint of a webpage. It turns everything on a page (text, images, elements) into objects that JavaScript can interact with. This lets anyone change what users see and how they interact with the page, like updating text or responding to button clicks.

How do you select an element from the DOM?

By id: document.getElementById('id') By class: document.getElementsByClassName('class') By tag: document.getElementsByTagName('tag') Or using CSS selectors: document.querySelector('selector') (for the first match) or document.querySelectorAll('selector') (for all matches).

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is when someone listen for an event (like a click) on a parent element, but handle it for its child elements. It's useful because it doesn't need to attach separate listeners to each child element, which saves memory and works great for dynamically added content.

How do you manipulate an element's attributes and styles using the DOM?

To change attributes: element.setAttribute('attr', 'value') or element.removeAttribute('attr'). To style: element.style.property = 'value' (like element.style.color = 'blue'). Or use classList to add/remove/toggle classes: element.classList.add('class').
